# Overview

This dataset was used in "An Inductive Logic Programming Approach for Entangled Tube Modeling in Bin Picking". This dataset is intended to be used to create theories (i.e. sets of rules in Prolog) that indicate how to connect a set of cylinders to form complete tubes.
